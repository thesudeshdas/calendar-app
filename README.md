## Run Locally

1 - Clone the project

```bash
  git clone https://github.com/thesudeshdas/calendar-app.git
```

2 - Go to the project directory

```bash
  cd calendar-app
```

3 - Install dependencies

```bash
  npm install
```

4 - Add environment variables

```bash
  REACT_APP_CLIENT_ID
  REACT_APP_DISCOVERY_DOC
  REACT_APP_API_KEY
```

5 - Start the server

```bash
  npm run start
```
